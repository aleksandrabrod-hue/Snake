# Snake
Projekt grupowy, gra wąż<br>
Napisany przez: **Aleksandra Bród** i **Zofia Szafran** <br>
Język: Python 3.13.5 <br>
Biblioteka: `pygame` 

## Opis

Prosta gra Snake, w której gracz steruje wężem i zbiera punkty.  
Celem jest zdobycie jak największej liczby punktów, unikając kolizji ze ścianami i własnym ogonem.


## Wymagania

- Python **3.12** (lub nowszy)
- System: Windows / Linux / macOS
- Git (opcjonalnie, ale zalecane)

## Instrukcja pobierania i instalacji

Poniżej opisane są wszystkie kroki potrzebne do pobrania, zainstalowania
i uruchomienia gry **Snake** od zera.

### 1. Pobranie projektu
- Pobierz repozytorium na swój komputer za pomocą polecenia:
```bash
git clone https://github.com/aleksandrabrod-hue/Snake.git 
```
- Następnie przejdź do katalogu projektu:
```bash
cd Snake
```
### 2. Środowisko wirtualne (venv)
Zalecane jest użycie środowiska wirtualnego, aby zainstalowane biblioteki
nie kolidowały z innymi projektami Pythona na komputerze.

Utwórz i aktywuj środowisko wirtualne:
```bash
python -m venv venv
source venv/Scripts/activate
```
Po poprawnej aktywacji w terminalu pojawi się oznaczenie (venv).

### 3. Instalacja wymaganych pakietów
Zainstaluj wszystkie potrzebne biblioteki zapisane w pliku requirements.txt:
```bash
pip install -r requirements.txt
```

### 4. Uruchomienie gry
```bash
python pro_gr.py
```
