# Snake
Projekt grupowy, gra wąż
Napisany przez: **Aleksandra Bród** i **Zofia Szafran** 
Język: Python 3.13.5  
Biblioteka: `pygame` 

## Opis

Prosta gra Snake, w której gracz steruje wężem i zbiera punkty.  
Celem jest zdobycie jak największej liczby punktów, unikając kolizji ze ścianami i własnym ogonem.
